# 好友 API

## `send_friend_poke` 发送好友戳一戳

### 参数

| 字段名 | 类型 | 描述 |
| --- | --- | --- |
| user_id | int64 | 好友 QQ 号 |
| is_self | int64 | 是否戳自己（默认值：`false`） |

### 返回值

此 API 无返回值。
## `send_profile_like` 发送名片点赞

### 参数

| 字段名 | 类型 | 描述 |
| --- | --- | --- |
| user_id | int64 | 好友 QQ 号 |
| count | int32 | 点赞数量（默认值：`1`） |

### 返回值

此 API 无返回值。
